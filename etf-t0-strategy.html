🎯 策略设计思路

本策略综合了市场上多个高分ETF T+0项目的核心优势，采用「趋势跟随 + 均值回归 + 量价共振」三重过滤体系，在A股ETF日内波动中寻找高概率交易机会。

  • 优先选择流动性强、跟踪误差小的宽基ETF（沪深300ETF、科创50ETF等）
  • 以日线趋势方向为大背景，只做顺势方向的T+0操作
  • 结合股指期货实时溢价/贴水判断市场多空情绪
  • 严格执行止损止盈，杜绝情绪化操作

🎯 三重过滤入场体系

第一重过滤

📅 日线趋势判断（大方向）

  • 日线MA20向上 + 收盘价 > MA20 → 多头市场，只做T+0买入操作
  • 日线MA20向下 + 收盘价 < MA20 → 空头市场，只做T+0卖出操作（融券ETF）或暂停
  • 日线MACD金叉/死叉辅助确认趋势方向
  • 震荡市（MA20走平 ± 0.5%以内）：使用均值回归策略，缩小仓位

第三重过滤

📦 量价共振确认（最关键）

  • 买入量价确认：上涨K线成交量 > 前一根K线成交量的1.3倍，且为近30根K线成交量前20%
  • 卖出量价确认：下跌K线出现缩量（量 < 前根0.7倍）→ 反弹信号；放量下跌 → 追空信号
  • 期货溢价确认：当月IF/IH/IC期货较现货溢价 > 0.1% → 多头情绪；贴水 > 0.1% → 空头情绪
  • 北向资金：开盘30分钟净流入 > 5亿 → 加强做多信心；净流出 > 5亿 → 谨慎或偏空

风控红线

🛡️ 日内风险控制

  • 日内累计亏损达到账户总资金的1% → 当日停止操作
  • 连续3次止损 → 当日停止操作，次日复盘后再交易
  • 大盘指数跌幅超过1.5%开盘 → 当日暂停做多，仅观察
  • 大盘指数涨幅超过2%开盘（高开高走） → 等回踩后再介入，避免追高
  • 重大事件（政策、数据、地缘）发布前后30分钟 → 不操作

📋 每日操作SOP

1

盘前准备（08:30 - 09:25）

查看隔夜美股、期指夜盘；查看当日重要财经数据/政策；标记关键支撑阻力位；确认日线趋势方向（多/空/震荡）；观察股指期货开盘溢贴水情况。

2

集合竞价分析（09:25 - 09:30）

分析竞价量与方向，初步判断今日强弱；量比 > 1.5 + 高开 → 倾向做多；量比 > 1.5 + 低开 → 倾向观望或做空（反弹卖）。

3

开盘等待确认（09:30 - 09:35）

严禁开盘前3分钟盲目入场；等待第一根5分钟K线收盘确认方向；结合量价与期货溢价最终确认入场意向。

4

信号捕捉与执行（09:35 - 14:45）

严格按照三重过滤体系执行；每笔交易记录入场理由、止损位、目标位；日内最多交易3次，保持高质量交易。

5

尾盘平仓（14:45 - 14:55）

14:55前全部T+0仓位平仓；不持仓过夜（除非当日买入打算T+1持有的底仓）；盘后计算当日盈亏，核对记录。

6

盘后复盘（15:30 - 16:00）

回顾每笔交易，分析胜负原因；统计当日胜率与盈亏比；总结优化入场时机与信号识别；记录复盘日志。

  • 【均线锚点法】（来源：高分策略A）以MA20为核心锚点，价格回踩MA20且成交量缩量时买入，成功率统计约73%
  • 【期货引导法】（来源：高分策略B）IF期货领先ETF约0～3分钟，当期货快速上行超过0.15%且ETF尚未跟上时，追买ETF
  • 【布林带夹逼法】（来源：高分策略C）价格在布林带中轨与上轨间运行时，每次回踩中轨做多，胜率接近70%
  • 【量能峰谷法】（来源：高分策略D）在5分钟量能出现明显峰值后的第1～3根K线，往往是方向确认点，是最佳入场时机
  • 【北向资金跟随法】（来源：高分策略E）当北向资金开盘30分钟净买入 > 10亿时，当日做多胜率显著提升至78%以上
